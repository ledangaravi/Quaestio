    Hypothesis 1

    Participants will display a 25% higher response rate when partaking in half-hour robot-led surveys as compared to human-led surveys

    Hypothesis 2

    The average participant will answer more short questions when interacting with a digital interface (animated face and voice/touch-screen input) as compared to a mechanical interface (mechanical face and button/slider input).

    Hypothesis 3

    What we do

    As the survey market is currently experiencing a rapid decline in participation, this project aims to explore the possibilities of improving survey participation through the utilisation of robots. Multiple experiments as well as a system design of a robot are proposed, with the aim of investigating the impact of human-robot interaction on survey response rates.

    Hardware

    Contains the mechanical face and input methods alongside the mobility related components

    User App

    This iOS/iPadOS app serves as the visual medium to conduct the surveys. The state controller takes in user inputs and controls the outputs.

    Database

    The database serves as a bridge between software and hardware, stores the configuration for the robot and holds all experimental data.

    Admin App

    Project team members can configure the robot and view the data through this iOS/iPadOS app.

    User Website

    Users can view and delete their own data. Users can also view the privacy policy and get in touch with the team.

    Animoji 

    Utilising a digital mask, the Animoji will mimic human expressions and facial movements when asking questions, as well as whilst waiting for the user to respond.
